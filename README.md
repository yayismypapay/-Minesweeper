# Learn Dock

*Learn Dock* is a simple study planner for students.  
It helps you keep your study tasks, schedule, notes, reminders, and focus time in one place.

*Learn Dock* — это простой планировщик для учёбы.  
Он помогает хранить расписание, домашние задания, заметки, напоминания и время фокуса в одном месте.

---

## What you can do / Что можно делать

- Create a weekly schedule  
  Создавать расписание на неделю

- Add and track homework  
  Добавлять и отслеживать домашку

- Write notes  
  Писать заметки

- Use a Pomodoro focus timer  
  Использовать Pomodoro-таймер для фокуса

- Add reminders  
  Добавлять напоминания

- Check study statistics  
  Смотреть статистику учёбы

- Switch between English and Russian  
  Переключать английский и русский язык

- Export and import your data  
  Экспортировать и импортировать свои данные

---

## Privacy / Приватность

Learn Dock does not need an account.  
Your data is saved only in your browser.

Learn Dock не требует аккаунт.  
Все данные сохраняются только в твоём браузере.

The app does not send your notes, homework, or schedule to any server.  
Приложение не отправляет заметки, домашку или расписание на сервер.

---

## How it works / Как это работает

This is a static website.  
It works without a backend, database, or API key.

Это статический сайт.  
Он работает без backend, базы данных и API-ключей.

Data is saved with `localStorage`.

Данные сохраняются через `localStorage`.
